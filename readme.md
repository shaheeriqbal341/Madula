# 🏥 Madula Bootstrap - Healthcare Website

A modern and responsive healthcare website built using **HTML, CSS, and Bootstrap 5**. The website provides a clean user interface for showcasing medical services, doctors, pricing plans, testimonials, and healthcare blogs.

---

## 🚀 Features

- ✅ Responsive Bootstrap 5 Design
- ✅ Modern Landing Page
- ✅ Medical Services Section
- ✅ About/Healthcare Information
- ✅ Qualified Doctors Section
- ✅ Pricing Table
- ✅ Patient Testimonials
- ✅ Medical Blog Section
- ✅ Clean UI & Smooth Layout
- ✅ Font Awesome Icons

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
